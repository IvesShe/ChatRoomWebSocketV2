# WebSocket聊天室(node.js + socket)

  ## Server端相關安裝
  ```node
  npm init -y
  npm i nodejs-websocket --save
  npm install nodemon -g  
  npm install --save-dev nodemon 
  nodemon
  ```
  
  ## 瀏覽器開啟客戶端(至少開2組以上比對使用)
  https://ivesshe.github.io/ChatRoomWebSocketV2/
